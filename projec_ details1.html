Vehicle Defects Management Application

Overview

In just 4 months, I led a team to build a vehicle defects management application that efficiently manages defects of all types of vehicles and human resources. The application was designed using React.js as the frontend stack and Spring Boot as the backend stack, while Redux was chosen as the state management tool.

The key features of the application included a 100% responsive design and a PWA (Progressive Web Application) feature that enhanced its usability across multiple devices.

Responsibilities

  • Ensured that the application met the requirements of the clients
  • Overseen the deployment process
  • Provided maintenance support post-release
