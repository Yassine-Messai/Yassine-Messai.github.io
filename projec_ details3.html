SocialPedia

Project Description

SocialPedia is a complete full-stack MERN application that allows users to share their thoughts, post images, and connect with friends. It includes authentication, user authorization, likes, comments, and a dark mode feature for a better user experience. This application was built using React, MongoDB, and Material UI.

Key Features:

  • User authentication and authorization using JSON Web Tokens (JWT)
  • Secure password hashing and encryption using bcrypt
  • User profile creation and editing
  • Ability to create, edit, and delete posts
  • Like and comment functionality for posts
  • Dark mode feature for improved user experience
  • Fully responsive design for mobile, tablet, and desktop devices

Technologies Used:

  • React.js for building the frontend user interface
  • Material UI for styling and design
  • MongoDB for storing and retrieving data
  • Express.js for building the backend API
  • Node.js for running the backend server
  • JSON Web Tokens (JWT) for user authentication and authorization
  • bcrypt for password hashing and encryption

Skills Utilized:

  • React.js
  • MongoDB
  • Express.js
  • Node.js
  • JSON Web Tokens (JWT)
  • bcrypt

Project Duration:

This project was completed in approximately 4 weeks.

Project Outcome:

socialPedia is a fully functional social media application with a responsive design and a variety of features that make it an attractive platform for users to share their thoughts and connect with others. This project helped to strengthen my skills in React, MongoDB, and backend development, as well as introduced me to important security concepts such as password encryption and user authentication using JSON Web Tokens (JWT).
